Here you can find the repo: Simple-Static-Site-Generator

Simple static site generator has one feature. Its algorithm is something like this:

  • traverses the directories and files
  • If directory contains header.html set header
  • If directory contains footer.html set footer
  • For each .html file in the directory
    • Concat the contents of header + file + footer
    • Save the result to the output directory
